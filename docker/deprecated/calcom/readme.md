# calcom

This project is a disaster. Utter cancer.
The setup here is only partially working - I still haven't gotten orgs and routing working right. At this point it's
very much not worth the hassle.

Taking 10 minutes to "build" a project written in an interpreted language is absolutely unacceptable.
The build requires more than 4GB of ram ... oh my literal god. "Set it to 16GB to avoid OOM" I BEG YOUR FINEST PARDON?!

Error messages are cryptic and useless - caused by simple misconfiguration issues that could have been validated instantly.
The secrets are baked into the docker images - you have got to be kidding me. Is it 2009?

Runs so fucking slow :/ Each click means looking at stupid NextJS empty placeholders until the app
populates ... AN EMPTY LIST!!!!
